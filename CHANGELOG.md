# Changelog

All notable changes to Facebook Register Tool will be documented in this file.

## [1.1.0] - 2026-05-05

### Added
- ✨ Lifetime license support (100 years expiry)
- 🎨 Modern license activation dialog with improved UX
- 🔧 License dashboard with real-time management
- 📊 Days remaining display for license expiry
- 🎯 Custom confirm/prompt dialogs in dashboard
- 🟡 Yellow Tool ID badge for better visibility
- 🔄 Auto-fix tool for converting 1-year licenses to lifetime

### Fixed
- 🐛 License expiry calculation (0 days now correctly = lifetime)
- 🐛 Close button icon color on hover (now white on red background)
- 🐛 Placeholder text visibility in license input field
- 🐛 Input mask validation issues
- 🐛 "No License" fallback display issue
- 🐛 Expiry date showing "Apr 11, 2126" instead of "Never (Lifetime)"

### Changed
- 🎨 Improved license dialog UI with better colors and spacing
- 🎨 Tool ID now has yellow background instead of gray
- 🚫 Removed annoying "License Required" popup on close
- 📝 Better placeholder text for ban reason input

### Removed
- ❌ Duplicate "License Required" message box
