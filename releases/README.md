# Releases Directory

This directory contains compiled releases of Facebook Register Tool.

## Release Naming Convention

```
FRT-v{version}-{platform}-{date}.{ext}

Examples:
- FRT-v1.1.0-windows-20260505.exe
- FRT-v1.1.0-windows-20260505.zip
```

## Release Structure

Each release should include:
- ✅ Compiled executable
- ✅ LICENSE file
- ✅ README with installation instructions
- ✅ CHANGELOG for that version
- ✅ Version number in filename
