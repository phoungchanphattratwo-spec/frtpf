## ១. ទិដ្ឋភាពទូទៅ

ឧបករណ៍ FRT គឺជាកម្មវិធីគ្រប់គ្រងគណនី Facebook ដែលមានមុខងារ៖
- ចុះឈ្មោះគណនី Facebook ស្វ័យប្រវត្តិ
- គ្រប់គ្រងគណនី និង ឧបករណ៍ Android ច្រើន
- ចូលគណនីស្វ័យប្រវត្តិ
- ស្វ័យប្រវត្តិកម្ម (Automation) ផ្សេងៗ
- ការ Seeding (ចែករំលែក, Like, Comment)
- គ្រប់គ្រង VPN និង Proxy

## ២. ផ្ទាំងគ្រប់គ្រង

ផ្ទាំងដំបូងដែលបង្ហាញស្ថានភាពទូទៅ។

### KPI Cards
| ឈ្មោះ | ការពិពណ៌នា |
|--------|------------|
| **សរុប** | ចំនួនគណនីសរុប |
| **សកម្ម** | គណនីដែលកំពុងប្រើ |
| **បញ្ហា** | គណនីដែលមានបញ្ហា |
| **ថ្ងៃនេះ** | គណនីបង្កើតថ្ងៃនេះ |

### ក្រាហ្វ
- **អត្រាជោគជ័យ** — ក្រាហ្វរង្វង់បង្ហាញ % ជោគជ័យ
- **ការបែងចែកគណនី** — ក្រាហ្វបង្ហាញប្រភេទគណនី (Email / Phone / ប្រភេទផ្ទាល់ខ្លួន)

### សកម្មភាពរហ័ស
- **នាំចូល** — បើកផ្ទាំងនាំចូលគណនី
- **បន្ថែមគណនី** — បើកផ្ទាំងនាំចូល
- **ធ្វើឱ្យស្រស់** — ធ្វើឱ្យស្រស់ស្ថិតិ
- **នាំចេញ** — នាំចេញទិន្នន័យ
- **លុប** — លុបកំណត់ហេតុ

### កំណត់ហេតុ (Activity Log)
- បង្ហាញសកម្មភាពទាំងអស់ក្នុងពេលជាក់ស្តែង
- ពណ៌ខុសគ្នាសម្រាប់ ✓ ជោគជ័យ, ✗ បរាជ័យ, ⚠ព្រមាន

### ការក្លែងបន្លំឧបករណ៍ (Device Spoofing)
- **បើកការក្លែងបន្លំ** — បើក/បិទ Spoofing
- **ប្តូររបៀប** — ជ្រើស Mode ការប្តូរ

- **Model ឧបករណ៍** — ប្តូរ Model
- **Build Fingerprint** — ប្តូរ Fingerprint
- **Serial** — ប្តូរ Serial Number
- **MAC Address** — ប្តូរ MAC Address
- **សាកល្បងការប្តូរ** — ពិនិត្យថាការប្តូរបានជោគជ័យ

### VPN
- **បើក VPN** — បើក/បិទ VPN
- **ប្រភេទ VPN** — ជ្រើស Mode (Select Location / Random)
- **Server VPN** — ជ្រើស Server
- **Username / Password** — ព័ត៌មានចូល VPN
- **តភ្ជាប់លើទូរស័ព្ទ** — ចាប់ផ្តើម VPN លើទូរស័ព្ទ
- **ផ្តាច់ពីទូរស័ព្ទ** — បញ្ឈប់ VPN

### ប៊ូតុងសំខាន់
- **រក្សាទុកការកំណត់** — រក្សាទុកការកំណត់ទាំងអស់
- **អនុវត្តការផ្លាស់ប្តូរ** — អនុវត្ត Spoofing ភ្លាមៗ
- **កំណត់រចនាសម្ព័ន្ធ** — បើក Dialog Advanced Config
- **ការកំណត់ស្វ័យប្រវត្តិ** — បើក Dialog Automation Settings

## ៥. គណនី និង ឧបករណ៍

ផ្ទាំងគ្រប់គ្រងគណនី Facebook និង ឧបករណ៍ Android ។

### តារាងគណនី
| ជួរ | ការពិពណ៌នា |
|-----|------------|
| # | លេខរៀង |
| UID | Facebook User ID |
| ស្ថានភាព | Active / Pending / Suspended / New |
| អ៊ីមែល | អ៊ីមែលចុះឈ្មោះ |
| ទូរស័ព្ទ | លេខទូរស័ព្ទ |
| ឈ្មោះ | ឈ្មោះ Facebook |
| ឧបករណ៍ | ឧបករណ៍ Android ភ្ជាប់ |
| Proxy | Proxy ដែលប្រើ |
| បង្កើត | ថ្ងៃបង្កើតគណនី |
| កំណត់ចំណាំ | កំណត់ចំណាំផ្ទាល់ |

### ការស្វែងរក និង ត្រង

- **ប្រភេទ** — ត្រងតាម All / Email / Phone / ប្រភេទផ្ទាល់ខ្លួន

- ត្រងគណនីតាមប្រភេទ

- **លុបកម្មវិធីថ្មីៗ** — Clear Recent Apps

## ១០. ការជួសជុលថ្មី

### ការជួសជុលក្នុងកំណែនេះ

#### 🔧 ប៊ូតុងបិទផ្ទាំង Import — Index ខុស
**បញ្ហា:** ប៊ូតុង × នៅផ្ទាំង Import ប្រើ `import_tab_index` ដែលជា Index ចាស់ ហើយអាចបិទផ្ទាំងខុស ប្រសិនបើ Tab ផ្សេងត្រូវបានបន្ថែមមុន។

**ការជួសជុល:** ប្រើ Widget Reference (`close_dynamic_tab_by_widget`) ជំនួស Index — ដូចដែល Auto Registration Tab ធ្វើស្រាប់។

---

#### 🔧 ផ្ទាំង "គណនី និង ឧបករណ៍" មិនបន្ត (Tab Not Persisting)
**បញ្ហា:** `account_tab = QWidget(self)` ធ្វើឱ្យ Widget ក្លាយជា Child របស់ MainWindow។ ពេល `showEvent` ដំណើរការ, `_hide_mystery_widgets` រកឃើញ Widget នេះ ហើយ Delete វា (`deleteLater()`) — ធ្វើឱ្យ Tab បាត់ពេល Startup។

**ការជួសជុល:** ប្តូរ `QWidget(self)` → `QWidget()` (គ្មាន Parent) ដូចដែល Import, Login, Automation Tabs ធ្វើ។

---

#### 🔧 Tab Name មិនបន្ត ពេលប្តូរភាសា
**បញ្ហា:** `save_settings` រក្សាទុកឈ្មោះ Tab ដែលបានបកប្រែ (ឧ. `"គណនី និង ឧបករណ៍"`) ប៉ុន្តែ `_open_one` ពិនិត្យតែ English String (`"Accounts and Devices"`) — ធ្វើឱ្យ Tab មិន Restore ពេល Restart។

**ការជួសជុល:**
- `save_settings` — Normalize ឈ្មោះ Tab ទៅ English ជានិច្ច
- `_open_one` — ទទួលស្គាល់ Translation ទាំងអស់ (EN, ខ្មែរ, VI)

---

#### 🔧 Active Tab ត្រឡប់ Tab ខុស ពេល Restart
**បញ្ហា:** `active_tab` ត្រូវបានរក្សាទុកជា Index លេខ (ឧ. `2` សម្រាប់ Settings)។ ពេល Restore, Dynamic Tabs ត្រូវបានបន្ថែមមុន ធ្វើឱ្យ Settings ផ្លាស់ទៅ Index `3` — ហើយ `setCurrentIndex(2)` ចូលទៅ Tab ខុស។

**ការជួសជុល:**
- `save_settings` — រក្សាទុក **ឈ្មោះ Tab** ជំនួស Index (ឧ. `"Settings"`)
- `_resolve_active_tab()` — ស្វែងរក Tab ដោយ Name ក្រោយ Restore ទាំងអស់ — ដំណើរការបានជានិច្ច មិនថា Tab ណាត្រូវបានបន្ថែម
