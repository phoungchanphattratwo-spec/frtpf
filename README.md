## 📋 Requirements

- **OS:** Windows 10/11 (64-bit)
- **Python:** 3.8+ (for development)
- **RAM:** 4GB minimum, 8GB recommended
- **Storage:** 500MB free space
- **Internet:** Required for license activation

## 🔧 Installation

### For Users (Compiled Version)

1. Download the latest release from [Releases](https://github.com/phoungchanphattratwo-spec/frtpf/releases)
2. Run the installer: `FRT-v1.1.0-windows-{date}.exe`
3. Follow the installation wizard
4. Launch the application
5. Activate with your license key

### For Developers

```bash
# Clone the repository
git clone https://github.com/phoungchanphattratwo-spec/frtpf.git
cd frtpf

# Install dependencies
pip install -r requirements.txt

# Run the application
python main.py
```

## 📖 Usage

### License Activation

1. Launch the application
2. Enter your license key in the format: `XXXX-XXXX-XXXX-XXXX`
3. Click "Activate"
4. The tool is now ready to use

### License Dashboard

Access the web-based license management dashboard:
1. Open `license_dashboard/index.html` in a browser
2. Connect to your Supabase instance
3. Manage licenses, users, and activity logs
